kubectl-neat-for-web

A browser-based tool to clean up noisy kubectl get ... -o yaml outputs.

🏠 Home 📋 Changelog

🔄 Changelog

v0.2.0 - 2024-12-15

✨ Added

  • 🔓 Secret decode support with stringData output
  • 📁 Upload entire folders containing YAML files
  • 🗜️ ZIP download for multiple processed files
  • ⚠️ Security warning for secret decoding feature

🔧 Changed

  • Improved UI for resource details and file operations
  • Enhanced fallback text processing when CDN resources are unavailable
  • Better handling of multi-document YAML files

🛠️ Fixed

  • Improved YAML parsing error handling
  • Better preservation of document structure in multi-file processing

v0.1.1 - 2024-11-30

✨ Added

  • 📄 File upload support for single and multiple YAML files
  • 📋 Copy to clipboard functionality
  • 📄 Download processed YAML as file

🔧 Changed

  • Enhanced UI with better mode switching
  • Improved responsive design for mobile devices

v0.1.0 - 2024-11-15

🚀 Initial Release

  • ✂️ Clean up verbose Kubernetes YAML outputs
  • 🛡️ Remove noisy fields: status, metadata.managedFields, metadata.creationTimestamp
  • 📎 Support for multi-document YAML with --- separators
  • 🛡️ 100% client-side processing - no backend dependency
  • ⚡ Instant preview in browser
  • 📱 Static HTML application - works offline
